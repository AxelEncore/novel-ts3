Если вы видите ошибку "User found: NO" при попытке входа,
это означает, что нужный пользователь не был создан в базе данных.

РЕШЕНИЕ:

1. ПРОСТОЙ СПОСОБ (рекомендуется):
   - Дважды щелкните на файл "add-user.bat" в папке проекта
   - Дождитесь завершения
   
2. ЧЕРЕЗ КОМАНДНУЮ СТРОКУ:
   npm run db:add-user

3. ПЕРЕСОЗДАТЬ ВСЮ БАЗУ:
   - Удалите файл database/encore_tasks.db
   - Дважды щелкните на run-migration.bat
